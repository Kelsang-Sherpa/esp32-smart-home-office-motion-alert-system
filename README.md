## Project Overview

This project implements a Smart Home Office Motion Alert System using the ESP32 Plus Development Board, PIR motion sensor, LCD display, and buzzer.

The system continuously monitors motion within a room. When human movement is detected, the PIR sensor triggers the ESP32 to activate the buzzer and display a warning message on the LCD screen. This provides both visual and audio notifications for security and monitoring purposes.

## Features

- Real-time motion detection using a PIR sensor
- LCD display for status and alert messages
- Buzzer alarm for audible notifications
- Embedded C++ programming using Arduino IDE
- Low-cost smart home security solution

## Hardware Components

- ESP32 Plus Development Board
- PIR Motion Sensor
- LCD Display
- Buzzer
- Jumper Wires
- Breadboard

## Software Used

- Arduino IDE
- ESP32 Board Package
- Embedded C++

## System Operation

1. ESP32 continuously reads the PIR sensor.
2. When motion is detected:
   - LCD displays "Motion Detected"
   - Buzzer activates
3. When no motion is present:
   - LCD displays "System Active"
   - Buzzer remains off
